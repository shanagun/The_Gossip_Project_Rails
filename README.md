# PROJET THP Semaine 5 jour 1 : The Gossip Project : Rails

## Objet du projet 
Développer un site avec rails 

### Pour lancer l'application

dans le dossier par défaut, exécuter :
$ bundle install

ensuite : 

$ rails db:create

ensuite : 

$ rails db:migrate

ensuite :

$ rails db:seed

et pour finir :

$ rails server
